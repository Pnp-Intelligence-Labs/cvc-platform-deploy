# Report HTML Output — Design Spec

Applies **only** to the HTML generated by `assemble_report()` in `report_builder.py`.
This is the published/downloaded report output. It does NOT apply to the platform UI.

## Brand: Plug and Play SLAM

SLAM = Supply Chain, Logistics & Advanced Manufacturing vertical.
Design is PnP brand-compliant but editorially flavored (not a pure corporate template).

## Colors

| Role | Hex | Usage |
|---|---|---|
| Navy | `#253B49` | Header background, footer background, h2 headings |
| Body text | `#313C51` | All paragraph text — never pure black |
| Background | `#F5F5F7` | Page background |
| SLAM teal | `#009EC2` | Accent: eyebrow line, sidebar border, citations, badge accents |
| Card bg | `#FFFFFF` | Section card backgrounds |

## Typography

- **Font:** `'Trebuchet MS', 'Gill Sans MT', sans-serif`
  - Trebuchet MS is PnP's approved cloud document font (Protipo is primary but commercial/not web-safe)
- Body: 0.975rem, line-height 1.7
- H1: 1.95rem, 700 weight, white (header only)
- H2: 1.2rem, 700 weight, navy `#253B49`

## Layout

- Full-width dark navy header: logo + SLAM eyebrow + title + meta line
- Body: max-width 900px, centered
- Section cards: white, `border-radius: 12px`, `box-shadow: 0 0 14px rgba(0,0,0,0.14)`
  - 12px radius = PnP brand spec (1/8" rounded corners)
  - 14% opacity / 0 offset / 14pt blur = PnP drop shadow spec
- Footer: dark navy, faded logo watermark + "Plug and Play SLAM · Confidential"

## Logo

- File: `api/static/pnp-slam-logo.png` (PLUGandPLAY wordmark, dark navy on white)
- Embedded as base64 data URI — report HTML is fully self-contained, works offline/downloaded
- Header: CSS `filter: brightness(0) invert(1); opacity: 0.92` → renders white on navy
- Footer: same filter + `opacity: 0.4` → subtle faded watermark

## Section type accents

- Tech Stack: left border SLAM teal `#009EC2`
- Investment Take: left border amber `#b45309`
- Deep Dive: left border blue `#0369a1`
- Sidebar pull-quote: float right, teal left border, light gray background
- Spotlight: green border, rounded card

## Citations

All citation colors use SLAM teal `#009EC2` (superscript, bracketed, MLA styles).

## Print

`@media print` ensures header/footer background colors are preserved (`print-color-adjust: exact`).
