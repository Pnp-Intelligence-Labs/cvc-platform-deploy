# CVC Platform Deploy

Private. Do not share.

Deployment guide, onboarding playbook, and replication toolkit for the CVC Intelligence platform.

## What This Repo Is

CVC Intelligence (`NEW-CVC-REPO`) is the platform codebase — built for CVC's internal team.
This repo is the **operator layer** — how to take that codebase and stand up a working instance for a new team.

## Contents (building out)

- `onboarding/` — step-by-step guide for a new team deployment
- `config/` — example environment configs, credential templates, seed data
- `scripts/` — install scripts, DB seed, role setup
- `docs/` — decisions, architecture notes, what to customize per team

## Status

Early stage. CVC deployment (Dell R620) is the reference implementation.
